## 1. НАЗК (Національне агентство з питань запобігання корупції)

### Базовий URL
`https://public-api.nazk.gov.ua/v2`

### Авторизація
❌ Не потрібна (відкритий API)

### Доступні endpoints
| Endpoint | Метод | Опис |
|----------|-------|------|
| `/documents/{document_id}` | GET | Отримати декларацію за UUID |
| `/documents/list` | GET | Пошук декларацій (filter + pagination) |
| `/countries/list` | GET | Довідник країн |

### Ключові параметри фільтрації
- `query` — текстовий пошук (3–255 символів)
- `user_declarant_id` — ID суб'єкта (1–10000000)
- `declaration_type` — тип декларації (1–4)
- `declaration_year` — рік (2015–поточний)
- `start_date` / `end_date` — UNIX timestamp
- `page` — сторінка (1–100)

### Оновлення 2020–2026
- **31.12.2020**: модернізація реєстру, `step_<0..17>`, поле `lastmodified_date` → `date`
- **2026 план**: детальна API документація для аналітиків

### Статус
✅ **Перевірено live 2026-03-03**: `HTTP 200`, response JSON

`https://bank.gov.ua/NBUStatService/v1/statdirectory`

### Авторизація
❌ Не потрібна

### Endpoints
| Endpoint | Параметри | Формат |
|----------|-----------|--------|
| `/exchange` | `json=true` | JSON (поточна дата) |
| `/exchange?date=YYYYMMDD` | `json=true` | JSON (історична) |
| `/exchange?valcode=usd&date=YYYYMMDD` | `json=true` | Конкретна валюта |

### Нове поле (з 21.01.2026)

### Статус
✅ **Перевірено live**: `HTTP 200`, response JSON

## 4. УБКІ (Українське бюро кредитних історій)

### 2 окремих API:

#### A) B2B кредитні звіти (XML)
- **URL**: `secure.ubki.ua/b2_api_xml/ubki/auth`
- **Формат**: XML (не JSON!)
- **Авторизація**: клієнтський договір УБКІ
- **Документація**: `wiki-ubki.atlassian.net`
- **Підтримка**: `support@ubki.ua`, `(044) 585-11-96`

#### B) ЄДРПОУ / публічні реєстри
- **URL**: `edrpou.ubki.ua/ua/api-integration`
- **2 шляхи доступу**:
  1. **Приват24 для бізнесу** — без договору, через «Каталог послуг → Досьє ділової репутації»
  2. **Пряме партнерство УБКІ** — підписання клієнтського договору

### Статус
✅ Live, XML-based, БЛОКЕР для прямого B2B: **договір УБКІ**

## 5. n8n — Automation Workflows

### Інтеграція
- **Метод**: HTTP Request Node (універсальний)
- Нативних «Ukrainian Gov» nodes **немає**
- Підтримується:
  - Імпорт curl-команд прямо в node
  - Авторизація через Header/Query/OAuth

### Приклад підключення НАЗК через n8n
```json
{
  "method": "GET",
  "url": "https://public-api.nazk.gov.ua/v2/documents/list",
  "params": { "query": "Іванов" },
  "authentication": "none"
}
```
